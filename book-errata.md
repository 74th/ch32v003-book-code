# 書籍の正誤表

## 第1版

### 14.1. CH32V003の省エネルギーモード

誤:

> スリープモード、スタンバイモードともに、「割込待ち」命令を実行すると各モードに移行します。

正:

> スリープモード、スタンバイモードともに、「割込待ち」命令を実行すると各モードに移行します。
> スタンバイモードの復帰後はクロック等がリセットされているため、初期化命令を呼び出す必要があります。

### 14.4 スタンバイモード WCH SDK

誤:

> スタンバイモードを利用したLチカは、スリープモードの実装における`__WFI()`の部分を、`PWR_EnterSTANDBYMode()`に置き換えるだけで実装できます。

正:

> スタンバイモードを利用したLチカは、スリープモードの実装における`__WFI()`の部分を、`PWR_EnterSTANDBYMode()`に置き換えます。
> スタンバイモード復帰後は、クロック等がリセットされているため、`SystemInit()`を呼び出す必要があります。

### 14.5 スタンバイモード ch32fun

誤:

> スタンバイモードから復帰した後は、設定を元に戻しておきましょう。
>
> ```c
> // 設定の解除
> NVIC->SCTLR &= ~((1 << 2) | (1 << 3));
> ```

正:

> スタンバイモードから復帰した後は、クロックの初期化を行い、設定を元に戻しておきましょう。
>
> ```c
> // クロックの初期化
> SystemInit();
> // 設定の解除
> NVIC->SCTLR &= ~((1 << 2) | (1 << 3));
> ```
